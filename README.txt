LANHOUSE OS — Guia Rápido
===========================

COMO ABRIR O SISTEMA:
1. Abra a pasta do projeto.
2. Dê dois cliques em "index.html"
3. O sistema abrirá no navegador.

PRIMEIRA VEZ / CONFIGURAÇÕES:
- Clique em "Configurações" (ícone de engrenagem).
- Preencha os dados da Lan House.
- Clique em "Salvar Configurações".
- Volte para o dashboard.

COMO IMPRIMIR OU SALVAR EM PDF:
1. Abra um módulo (Currículo ou Recibo).
2. Preencha os dados.
3. Use o botão do módulo:
   - Currículo: "Imprimir / Salvar PDF"
   - Recibo: "Imprimir" ou "Exportar PDF" (ambos chamam a impressão do navegador)
4. Na janela de impressão, escolha:
   - "Salvar como PDF" na lista de impressoras.

ATENÇÃO (OFFLINE / DADOS NO NAVEGADOR):
- Este sistema é 100% offline (sem dependências de CDN em produção).
- As configurações e rascunhos ficam salvos apenas no `localStorage` deste navegador neste computador.
- Se você limpar histórico/cache do navegador, os rascunhos salvos podem ser apagados.

RAScunho do CURRÍCULO:
- O módulo de Currículo salva automaticamente/por ação o rascunho com a chave "curriculo_rascunho".
- Esse rascunho inclui campos dinâmicos (experiências, formações e cursos).
